## 1. 專案概述

### 1.1 專案定位

一套針對**小學六年級拔尖學生**的記敘文寫作增潤電子教材，以單頁 HTML 網頁形式呈現，用於**一對一私人補習**場景。配合**香港小學中國語文課程**框架，聚焦記敘文寫作能力提升，並以描寫技巧作為核心支撐。

### 1.2 核心目標

- 幫助已有寫作基礎的學生，從「寫得對」提升到「寫得好」
- 提供系統化的寫作技巧框架，非零散知識點堆砌
- 讓補習老師有一份隨開即用的完整教學工具

### 1.3 設計原則

| 原則 | 說明 |
|------|------|
| **極簡部署** | 一個 HTML 檔案 + 圖片資料夾，瀏覽器直接打開即可使用 |
| **補習場景優先** | 所有互動設計圍繞「老師引導 + 學生動手」的 1v1 節奏 |
| **三段式教學** | 每個單元統一：知識點 → 範文賞析 → 實戰練習 |
| **零外部依賴** | 無需伺服器、無需資料庫、無需 npm install |

### 1.4 約束條件

- 從零開始創作，無現有素材可沿用
- 不需過於複雜的平台功能
- 圖片使用網上搜尋的卡通插圖（簡潔線條、暖色調風格）
- 桌面和 iPad 均可舒適使用

## 2. 內容架構

### 2.1 單元總覽

全教材共 **6 個教學單元**，由基礎到進階，逐步建構學生的記敘文寫作能力。

```
單元一：記敘文的骨架（基礎）     →  六要素 + 敘事結構
單元二：鳳頭 — 抓人的開頭        →  四種開頭策略
單元三：豬肚 — 把事情寫具體      →  描寫技巧核心（重點密集區）
單元四：豹尾 — 結尾的藝術        →  四種結尾策略
單元五：語言的打磨               →  修辭 + 句式節奏
單元六：從好到精                 →  立意提升 + 綜合實戰
```

### 2.2 單元一：記敘文的骨架 — 六要素與敘事結構

**📖 知識點**
- 記敘文六要素：時間、地點、人物、起因、經過、結果
- 三種敘事順序：
  - 順敘（最穩妥，考試安全牌）
  - 倒敘（先說結局，製造懸念）
  - 插敘（補充背景，豐富層次）
- 如何選擇敘事順序：考試 vs 創作的策略差異

**🔍 範文賞析**
- 範文：《運動會的一天》（約 400 字校園生活短文）
- 六要素顏色標註（每種要素對應一種顏色）
- 結構拆解圖：開頭 → 發展 → 高潮 → 結尾

**✏️ 實戰練習**
- 練習一（填空）：讀短新聞報導，找出六要素
- 練習二（排序）：雜亂事件句重組為順敘
- 練習三（改寫）：順敘短文改寫為倒敘版本

---

### 2.3 單元二：鳳頭 — 寫一個抓人的開頭

**📖 知識點**
- 四種開頭策略（各附完整例句）：
  1. 開門見山式：「那天，我做了一件至今難忘的事。」
  2. 懸念式：「如果你只有一天的生命，你會做什麼？這個問題，我在小六那年找到了答案。」
  3. 場景式：「陽光穿過窗簾的縫隙，在地板上畫出一道金色的線⋯⋯」
  4. 對話式：「『你真的要去嗎？』媽媽的聲音在背後響起。」
- 每種策略的適用場合與效果分析

**🔍 範文賞析**
- 同一主題《第一次上台演講》，展示四種不同開頭
- A/B 對比：好開頭 vs 平庸開頭（同一事件，直觀感受落差）

**✏️ 實戰練習**
- 練習一（配對）：5 個開頭句子 → 判斷類型
- 練習二（改寫）：2 個平凡開頭 → 改寫為精彩版本
- 練習三（創作）：指定題目，試寫 2 種不同開頭

---

### 2.4 單元三：豬肚 — 把事情寫具體（⭐ 重點密集單元）

**📖 知識點**

> ⭐ **核心重點一：心理描寫三層次**
> 表層感受 → 內心獨白 → 情感變化
> - 表層：「我很緊張。」（一句帶過）
> - 獨白：「『萬一忘詞怎麼辦？』這個念頭在腦中盤旋⋯⋯」
> - 變化：從緊張到放鬆、從害怕到勇敢，寫出心理的轉折弧線

> ⭐ **核心重點二：五感描寫法**
> 視覺、聽覺、嗅覺、味覺、觸覺的調用
> - 不只寫「看到」的，調動全部感官豐富畫面
> - 例：寫雨 — 看見雨絲斜斜落下（視）、聽見屋簷滴答作響（聽）、聞到泥土濕潤的氣味（嗅）

- 動作分解術：把「他哭了」寫成 80 字的過程
- 對話穿插技巧：
  - 直接引語 vs 間接引語的選擇
  - 對話如何推動情節（不只為對話而對話）

**🔍 範文賞析**
- 對比展示：「乾癟版」（100字）vs「豐滿版」（350字）— 同一事件
- 豐滿版中各類描寫技巧標註位置
- 彩蛋：真實學生原作 vs 修改後對比

**✏️ 實戰練習**
- 練習一（擴寫）：把「他很生氣地走了」擴寫為 100 字
- 練習二（填空）：給場景框架，補入五感描寫
- 練習三（改造）：沉悶間接引語 → 生動直接對話

---

### 2.5 單元四：豹尾 — 結尾的藝術

**📖 知識點**
- 四種結尾策略：
  1. 呼應開頭式：回到開頭的畫面或問題，形成閉環
  2. 感悟昇華式：從具體事件提煉出人生體悟
  3. 留白想像式：不把話說盡，留給讀者回味空間
  4. 意外反轉式：結尾出現意料之外的轉折
- 結尾四大忌：草草收場 / 無限延伸 / 硬加道理 / 偏離主題

**🔍 範文賞析**
- 精選 4 篇短文結尾，逐一點評高下
- 同一篇文章配 4 種不同結尾，比較效果

**✏️ 實戰練習**
- 練習一（診斷）：判斷 4 個結尾好壞，說出理由
- 練習二（配對）：給文章主體，選最佳結尾
- 練習三（創作）：給另一篇主體，自己寫結尾

---

### 2.6 單元五：語言的打磨 — 修辭與句式（⭐ 重點密集單元）

**📖 知識點**

> ⭐ **核心重點三：四大常用修辭**
> - 比喻（明喻：「像⋯⋯一樣」/ 暗喻：「是⋯⋯」）
> - 擬人（賦予事物人的特質）
> - 排比（相似句式重複，增強氣勢）
> - 誇張（刻意放大以突出效果）

> ⭐ **核心重點四：進階技巧**
> - 通感（感官互通）：「她的聲音像蜜糖一樣甜」（聽覺 → 味覺）
> - 對比襯托：用黑夜襯托星光，用懦弱襯托勇敢

- 句式節奏：長句營造氣氛 vs 短句製造張力、長短交錯的韻律
- 常見語病診斷：重複用詞、陳詞濫調、「然後病」

**🔍 範文賞析**
- 摘選朱自清《背影》、魯迅《風箏》片段，賞析大師筆法
- 對比：「學生腔」vs「文學語言」（同一意思，不同表達的差距）

**✏️ 實戰練習**
- 練習一（辨識）：讀片段，圈出修辭手法
- 練習二（仿寫）：給名家句子結構，換主題仿寫
- 練習三（潤色）：5 句「學生腔」句子，逐句打磨

---

### 2.7 單元六：從好到精 — 立意提升與綜合實戰（⭐ 重點密集單元）

**📖 知識點**

> ⭐ **核心重點五：選材的視角**
> 同一個題目，為什麼有人寫得特別？
> — 選材角度決定高度
> 例：《一件小事》—
>   普通選材：撿到錢包交還（千篇一律）
>   獨特選材：一個微小舉動引發的自我反省（真實動人）

> ⭐ **核心重點六：主題提煉三步法**
> 事件 → 衝突/轉折 → 感悟/成長
> - 不只寫「發生了什麼」
> - 要寫「這件事如何改變了我」
> - 記敘文的靈魂在於：從經歷中找到意義

**🔍 範文賞析**
- 同題異作對比：《一件小事》普通版 vs 優秀版
- 拆解優秀版的選材角度和立意層次

**✏️ 實戰練習**
- 練習一（腦力激盪）：題目《那一刻，我長大了》，引導不同選材角度
- 練習二（命題寫作）：3 選 1 題目，完整寫作
- 練習三（自評）：對照檢核表自我檢查作品

---

### 2.8 附加資源：寫作錦囊

放置於頁面最底部，作為快速參考工具：

- **⚡ 急救包**：考試前 10 分鐘可看的重點濃縮卡
- **📝 好詞好句庫**：分類整理（寫人 / 寫景 / 寫事 / 寫情）
- **✅ 自評檢核表**：通用版，適用任何記敘文，包含：
  - 六要素是否完整？
  - 開頭是否吸引人？
  - 是否有具體描寫（五感/動作/心理）？
  - 結尾是否有力度？
  - 語言是否生動（修辭/句式）？
  - 立意是否清晰？

### 3.2 視覺風格

| 元素 | 設計決策 |
|------|----------|
| **色調** | 主色：墨藍 / 藏青色（沉穩學術感）；點綴：暖金色（重點強調），適合小六接近中學的成熟度 |
| **標題字體** | 思源宋體（Noto Serif TC），營造「中文課本」質感 |
| **內文字體** | 思源黑體（Noto Sans TC），確保長文閱讀舒適 |
| **Fallback 字體** | 標題 → 標楷體 / 新細明體；內文 → 微軟正黑體 / 蘋方 |
| **圖片** | 卡通插圖（簡潔線條、暖色調），每個單元一張主題圖 + 頁籤小圖標 |
| **⭐ 核心重點** | 金色左邊框 + 淡金底色 + 「重點」標籤徽章，與一般內容明確區分 |
| **響應式** | CSS media query：桌面（≥768px）完整雙欄；平板/手機（<768px）單欄堆疊 |

### 3.3 三種內容區塊的視覺語言

| 區塊 | 佈局方式 | 視覺特徵 |
|------|----------|----------|
| **📖 知識點** | 卡片式（card layout） | 重點用邊框引號框出、關鍵詞粗體+底色標記、⭐重點區塊有金色邊框 |
| **🔍 範文賞析** | 桌面：左原文右旁注雙欄；平板/手機：上下排列 | 原文高亮可點擊，點擊後右側/下方彈出賞析筆記 |
| **✏️ 實戰練習** | 題目卡片 + 輸入區 + 答案按鈕 | 答案預設隱藏，點擊按鈕揭示；引導提問用對話氣泡樣式 |

### 3.4 「口頭討論提示卡」設計

每個知識點旁附小型對話氣泡 💬，內容為 1-2 句引導提問，協助老師發問：

> 💬 *「你覺得這段描寫，哪一句最能讓你『看到』畫面？」*

視覺樣式：淺色背景圓角框 + 左側小圖標，與正文卡片區分。

## 4. 互動功能設計

### 4.1 頁籤切換系統

- 每個單元內三個頁籤：知識點 / 範文賞析 / 實戰練習
- 預設開啟「知識點」頁籤
- 切換時平滑動畫過渡（CSS transition）
- 同一時間每個單元只顯示一個頁籤內容

### 4.2 功能清單

| 功能 | 應用場景 | 行為描述 |
|------|----------|----------|
| **頁籤切換** | 每個單元內 | 點擊頁籤按鈕切換顯示內容區 |
| **高亮標註** | 範文賞析 | 點擊範文中句子 → 自動高亮 + 右側彈出賞析筆記 |
| **手風琴展開** | 知識點 | 核心概念先展示，點擊展開詳細說明/更多例子 |
| **對比切換** | 範文賞析 | 「原版」⇄「潤色版」一鍵切換，直觀對比 |
| **填空作答** | 實戰練習 | `<textarea>` 直接輸入，點「對答案」顯示參考答案 |
| **答案揭示** | 所有練習 | 參考答案預設隱藏（`display:none`），點擊按鈕揭示 |
| **進度標記** | 單元標題旁 | ☐ → ☑ 點擊打勾，狀態存入 localStorage |
| **返回頂部** | 全局右下角 | 固定浮動按鈕，點擊平滑滾動回頁面頂部 |

### 4.3 補習場景特殊設計

- **引導提問卡**：每個知識點附帶，幫助老師發問
- **進度記憶**：localStorage 記錄已完成的單元，下次打開頁面還原打勾狀態
- **無干擾設計**：沒有彈窗廣告、沒有自動播放、沒有計時器壓力

## 5. 技術實作方案

### 5.1 檔案結構

```
narrative-writing/
├── index.html           ← 唯一核心檔案（內嵌所有 CSS + JS）
└── images/
    ├── unit-1.png       ← 單元一主題插圖
    ├── unit-2.png       ← 單元二主題插圖
    ├── unit-3.png       ← 單元三主題插圖
    ├── unit-4.png       ← 單元四主題插圖
    ├── unit-5.png       ← 單元五主題插圖
    ├── unit-6.png       ← 單元六主題插圖
    ├── icon-knowledge.png  ← 知識點頁籤圖標
    ├── icon-reading.png    ← 範文賞析頁籤圖標
    ├── icon-practice.png   ← 實戰練習頁籤圖標
    └── icon-star.png       ← 重點標記圖標
```

### 5.2 技術選型

| 層面 | 方案 | 說明 |
|------|------|------|
| **結構** | 純 HTML5 語義標籤（`<section>`, `<article>`, `<nav>`） | 無框架依賴 |
| **樣式** | 內嵌 CSS3（`<style>` 標籤內） | 使用 CSS custom properties 管理顏色變數 |
| **互動** | 原生 JavaScript ES6（`<script>` 標籤內） | 無 jQuery、無框架 |
| **字體** | Google Fonts CDN（`<link>` 載入 Noto Serif TC + Noto Sans TC） | 離線時回退系統中文字體 |
| **圖片** | PNG 本地檔案 | 離線可用，無外部 CDN 依賴 |
| **儲存** | localStorage | 僅用於進度標記，無隱私敏感資料 |

### 5.3 核心 JavaScript 模組（預計 < 300 行）

### 5.4 CSS 架構（預計 < 500 行）
